# Portfolio — Sergio Durán Utrera

CV interactivo / portafolio personal de página única, desplegado en GitHub Pages.

## Stack

- HTML semántico
- SCSS (arquitectura ITCSS + BEM)
- JavaScript vanilla (i18n ES/EN)

## Desarrollo

```bash
npm install      # instala sass
npm run build    # compila scss/main.scss -> css/main.css
npm run dev      # recompila al guardar (watch)
```
